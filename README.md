# 🎓 Student Performance Analysis

## 📌 Objective
This project analyzes student performance and explores factors that may influence academic outcomes.

## 📊 Dataset
- Student Performance Dataset
- Contains features such as study time, absences, family background, and final grades (G3)

## 🔍 Analysis
- Distribution of final grades (G3)
- Relationship between study time and performance

## 📈 Key Findings
- Most students score between 8 and 15, indicating an average performance level
- Study time shows a positive but moderate relationship with final grades

## 🛠 Tools Used
- Python
- pandas
- matplotlib

## 🚀 Future Work
- Add machine learning models to predict student performance
- Explore more factors such as absences and lifestyle habits
